INTRODUCTION:
------------

This module contains script files to get hosts.


REQUIREMENTS:
------------

This module requires the following modules:

 * Python 3
   Libraries
  * requests
  * sys
  * json
  * time

 * The scripts must be run outside sddc-manager environment.

 * DNS resolution must be done for sddc-manager.

USAGE:
-----
Usage:	python get_hosts.py <hostname> <username> <password>
